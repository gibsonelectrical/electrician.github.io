HTML Structure

The template is based on Bootstrap 3 Framework - http://getbootstrap.com

<div class="row">
  <div class="col-md-12">
        Level 1 column
        <div class="row">
            <div class="col-md-6">Level 2</div>
            <div class="col-md-6">Level 2</div>
        </div>
    </div>
</div>
Please note: You can use all Classes and Scripts from Bootstrap 3, they will all work! Not only the grid!

Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases. If you need more information about the Main Grid, please visit this site: http://getbootstrap.com/css/#grid

Customization and Styles

General

This Theme has various Style options for different Kinds of Layouts.

Colors

Check the /colors folder of the css folder. You can include one of our predefined Stylesheets into the Head Section of the Theme, right after the style.css file. If the color you need to use is not within one of default color files, you can just edit it and replace the occurences of main color with your own color code (for fast results, use the Find and replace function of your text editor)

Fonts

To use another Font, please choose your new Font at the Google Font Site, include the given Style Tag inside the Head Section of the Theme and add the CSS Markup to style.css file.

In the default Theme we have used this Code for example:

<link href='http://fonts.googleapis.com/css?family=Source+Sans+Pro%3A400,700%3Alatin%7CMontserrat%3A700%3Alatin"' rel='stylesheet' type='text/css'>

Layout

The Main Layout of the Theme can easily be changed by adding or removing one CSS Class to the Body.

Use this for the Boxed Version:

<body class="boxed">
    ...
  
</body>

Backgrounds (for boxed Version only)

You can define the Background for the Boxed Version by adding a path to the Body style in style.css file.

Available default backgrounds are provided within images/backgrounds folder of the theme.

Example:

body {background-image:url('../images/backgrounds/1.png')}

Menus

Buildpress is using the default Bootstrap menu system with some changes to match the template style.

The menu type can be easily switched from regular to sticky type. Use this code for the sticky menu variation:

<body class="fixed-navigation">
	...
 
</body>

To customize the maps simply check the contact-us.html file for the markup. To edit the marker just replace the image - images/map_pin.png.

Pages

We have added a lot of Sample Pages to give you some Inspiration how your Content maybe could look like. Please see the different HTML Pages.

Dynamic Contact Form

To use the dynamic contact form which will deliver e-mails right to the specified mailbox, you have to make sure your server supports PHP <= 5.2. The required steps are the following:

  1. Change all the links to point to contact-us.php file instead of contact-us.html
  2. Edit the file contact/process-form.php. On line 64, change email@domain.com to your own e-mail address.
  3. (Optional) Check lines 68 and 72 for success and error messages texts
  4. (Optional) The lines 62 can be safely removed if you don't want the sender's name to be explicitly included in the message body

RTL MODE

To use RTL mode, just include the rtl.css file in the /css folder right after your style.css file declaration. Example: 

<link rel="stylesheet" type="text/css" href="css/rtl.css"/>
